Simulation Schema¶

This project describes two types of simulations:

  • Atomic: A single action taken in order to achieve a particular goal.

  • Campaign: An organized course of action composed of a series of steps to achieve a goal.

Schemas¶

  • Atomic Template Format

  • Campaign Template Format

Atomic Template Format¶

In its simplest structure, an atomic template has the following elements:

schema: "string"
id: "string"
name: "string"
metadata:
  creationDate: "string"
  modificationDate: "string"
  description: "string"
  contributors:
    - "string"
  mitreAttack:
    - technique: "string"
      tactics:
        - "string"
authorization:
  - resource: "string"
    permissionsType: "string"
    permissions:
      - "string"
execution:
  type: "string"
  platform: "string"
  executor: "string"
  parameters:
    "string":
      type: "string"
      description: "string"
      required: bool
      defaultValue: "string"

  modificationDate: "string"
